## 介紹 
這是經典的俄羅斯方塊遊戲，用C++寫，map最大為45x45，共有19種俄羅斯方塊，包含L、T、I、O、J、S及其不同方向(用1、2、3、4表示)
## 操作
每次輸入一行 -> L4 8 0 代表 形狀為L，旋轉代號4、從第8行掉落、第一次碰到物體後，向右移動0格
## 實作 
用二維陣列存放map，陣列內為0表示空、陣列內為1表示有方塊，並用for迴圈讓方塊一格一格掉落或左右移動。<p>
